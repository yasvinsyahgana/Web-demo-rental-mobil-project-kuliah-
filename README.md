# 🚗 Rental Mobil Web App

## 📌 Deskripsi

Aplikasi web untuk menyewa mobil dengan fitur login, dashboard, dan transaksi penyewaan.

## 🎯 Fitur

* Login multi user (admin & customer)
* Daftar mobil
* Sewa & pengembalian
* Cetak struk

## 🛠️ Teknologi

* HTML
* CSS
* JavaScript
